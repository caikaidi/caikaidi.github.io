光束质量自动化

发表于 2021-03-10 更新于 2021-12-15 分类于 工具 阅读次数： Valine：

通过读取CCD的bgdata文件直接处理得到光束质量$M^2$，将一个小时的工作流程直接快进到一秒。

完整代码放在GitHub仓库，这里简单说一下原理。
在没有光束质量分析仪的情况下，光束质量$M^2$通常使用CCD测量高斯光束束腰前后光斑来进行拟合得到。
我们使用Spiricon的CCD，其数据文件*.BGdata使用h5py格式保存，可以通过这个库读取：

（vim打中文真的不舒服，还是用英语写吧）

Those data is organized linearly.
For a CCD of 320 x 320 resolution, there are 102400 data linearly.

To deal with these data, I dig deep through the iso standard of beam quality.
There are 3 steps to go.
First, get the interest area from the whole CCD.
Second, calculate the $d4_\sigma$ radius of the beam.
Last, fit a formula to get the $M^2$.

Interest area should be about 3 times the beam area, as ISO standard required.
The way I do this is get $d4_\sigma$ radius of the whole CCD and use this area as interest area, calculate the radius again.
So it’s like:

The calculation of $d4_\sigma$ is relatively easy by just do a integral (summary):
$$
d4_\sigma = 4\sigma= 4\sqrt{\frac{\int_{-\infty}^{\infty}\int_{-\infty}^{\infty}I(x,y)(x-\overline{x})^2dxdy}{\int_{-\infty}^{\infty}\int_{-\infty}^{\infty}I(x,y)dxdy}}
$$

Noise reduction is very important here.
I make a parameter to kill all data below some value based on the peak intensity.
For example set noise as 1% of the peak intensity, all intensity below this value will be set as 0.
